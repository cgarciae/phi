phi module

Phi

Python is a very nice language that favors readability but its not very strong at functional programming and this often leads to repetitive code. Phi intends to remove as much of the pain as possible from your functional programming experience in Python by providing the following modules

  • dsl: a small DSL that helps you compose computations in various ways & more.
  • lambdas: easy way to create quick lambdas with a mathematical flavor.
  • builder: an extensible class that enables you to integrate other libraries into the DSL through a fluent API.
  • patch: this module contains some helpers that enable you to integrate a complete existing module or class into the DSL be registering its methods/functions into a Builder.

Getting Started

The global P object exposes most of the API and preferably should be imported directly. The most simple thing the DSL does is function composition:

from phi import P

def add1(x): return x + 1
def mul3(x): return x * 3

x = P.Pipe(
    1,     #input 1
    add1,  #1 + 1 == 2
    mul3   #2 * 3 == 6
)

assert x == 6

Use phi lambdas to create the functions

from phi import P

x = P.Pipe(
    1,      #input 1
    P + 1,  #1 + 1 == 2
    P * 3   #2 * 3 == 6
)

assert x == 6

Create a branched computation instead

from phi import P

[x, y] = P.Pipe(
    1,  #input 1
    [
        P + 1  #1 + 1 == 2
    ,
        P * 3  #1 * 3 == 3
    ]
)

assert x == 2
assert y == 3

Compose it with a multiplication by 2 (P * 2)

from phi import P

[x, y] = P.Pipe(
    1,  #input 1
    P * 2,  #1 * 2 == 2
    [
        P + 1  #2 + 1 == 3
    ,
        P * 3  #2 * 3 == 6
    ]
)

assert x == 3
assert y == 6

Give names to the branches

from phi import P

result = P.Pipe(
    1,  #input 1
    P * 2,  #1 * 2 == 2
    dict(
        x = P + 1  #2 + 1 == 3
    ,
        y = P * 3  #2 * 3 == 6
    )
)

assert result.x == 3
assert result.y == 6

Divide the x by the y.

from phi import P, Rec

result = P.Pipe(
    1,  #input 1
    P * 2,  #1 * 2 == 2
    dict(
        x = P + 1  #2 + 1 == 3
    ,
        y = P * 3  #2 * 3 == 6
    ),
    Rec.x / Rec.y  #3 / 6 == 0.5
)

assert result == 0.5

Save the value from the P * 2 computation as s and load it at the end in a branch

from phi import P, Rec

[result, s] = P.Pipe(
    1,  #input 1
    P * 2, {'s'}  #2 * 1 == 2, saved as 's'
    dict(
        x = P + 1  #2 + 1 == 3
    ,
        y = P * 3  #2 * 3 == 6
    ),
    [
        Rec.x / Rec.y  #3 / 6 == 0.5
    ,
        's'  #load 's' == 2
    ]
)

assert result == 0.5
assert s == 2

Add 3 to the s loaded s... because you can

from phi import P, Rec, Read

[result, s] = P.Pipe(
    1,  #input 1
    P * 2, {'s'}  #2 * 1 == 2, saved as 's'
    dict(
        x = P + 1  #2 + 1 == 3
    ,
        y = P * 3  #2 * 3 == 6
    ),
    [
        Rec.x / Rec.y  #3 / 6 == 0.5
    ,
        Read.s + 3  # 2 + 3 == 5
    ]

assert result == 0.5
assert s == 5

Use the Write object instead of {...} just because

from phi import P, Rec, Read, Write

[result, s] = P.Pipe(
    1,  #input 1
    P * 2, Write.s  #2 * 1 == 2, saved as 's'

    ,
        y = P * 3  #2 * 3 == 6

    [
        Rec.x / Rec.y  #3 / 6 == 0.5
    ,
        Read.s + 3  # 2 + 3 == 5
    ]
)

assert result == 0.5
assert s == 5

Add an input Val of 9 on a branch and add to it 1 just for the sake of it

from phi import P, Rec, Val

[result, s, val] = P.Pipe(
    1,  #input 1
    P * 2, Write.s  #2 * 1 == 2, saved as 's'
    dict(
        x = P + 1  #2 + 1 == 3
    ,
        y = P * 3  #2 * 3 == 6
    ),
    [
        Rec.x / Rec.y  #3 / 6 == 0.5
    ,
        Read.s + 3  # 2 + 3 == 5
    ,
        Val(9) + 1  #input 9 and add 1, gives 10
    ]
)

assert result == 0.5
assert s == 5
assert val == 10

def Context(

cls, *args)

Builder Core. Also available as a global function as Context.

Returns the context object of the current dsl.With statemente.

Arguments

  • *args: By design Context accepts any number of arguments and completely ignores them.

This is a classmethod and it doesnt return a Builder/Lambda by design so it can be called directly:

from phi import P, Context, Obj

def read_file(z):
    f = Context()
    return f.read()

lines = P.Pipe(
    "text.txt",
    P.With( open,
        read_file,
        Obj.split("\n")
    )
)

Here we called Context with no arguments to get the context back, however, since you can also give this function an argument (which it will ignore) it can be passed to the DSL so we can rewrite the previous as:

from phi import P, Context, Obj

lines = P.Pipe(
    "text.txt",
    P.With( open,
        Context, # f
        Obj.read()
        Obj.split("\n")
    )
)

Context yields an exception when used outside of a With block.

def ThenAt(self, n, expr, *args, **kwargs):
    _return_type = None
    if '_return_type' in kwargs:
        _return_type = kwargs['_return_type']
        del kwargs['_return_type']
    def _lambda(x):
        x = self(x)
        new_args = args[0:n] + (x,) + args[n:] if n >= 0 else args
        return expr(*new_args, **kwargs)
    return self.__unit__(_lambda, _return_type=_return_type)
